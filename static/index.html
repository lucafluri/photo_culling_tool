Shortcuts — left-hand, ESDF layout

Navigate

  • ESDF move (or arrows)
  • Home/End first / last
  • G / Enter loupe view
  • Esc back / close
  • T cycle filter
  • V this help

Mark

  • W keep  R reject  Q clear flag
  • 1–5 stars  ` 0 stars
  • flags auto-advance to next

Compare

  • Space add/remove from compare set
  • C open compare (set, or current+next)
  • X clear compare set
  • SDF (≤3 in compare) keep that one, reject rest, close
  • >3 selected → vote ←/S vs →/F to find the best

Loupe / Compare

  • scroll / ED zoom (synced)
  • drag pan (synced in compare)
  • SF prev / next (loupe)
  • Z / dblclick reset zoom
  • marking works here too

Edit (loupe)

  • edit bar: rotate, flip, crop, tone
  • double-click a slider to reset it
  • Ctrl+Z / Ctrl+Y undo / redo
  • non-destructive — Reset reverts

Mouse mode

  • B toggle mouse mode
  • hotkeys act on the hovered photo
